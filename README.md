## Currently Building

- **TakeOff**: a native travel companion with atlas, trips, budgets, document vault, clocks, currency, widgets, and optional iCloud sync.
- **Artifact**: an iOS and Mac Catalyst viewer for AI-agent-generated HTML, Markdown, TSX, JSX, SVG, and Mermaid files.
- **Tiles**: a local-first workout and route mapping app for seeing where you have explored over time.
- **PromoKit**: an open-source SwiftUI app for organizing App Store offer-code campaigns.

## Working Style

I like apps that feel native, keep user data close to the device, and make complicated workflows feel direct. Most of my current projects use SwiftUI, SwiftData, CloudKit, MapKit, WidgetKit, StoreKit, and small static websites for launch/support pages.
